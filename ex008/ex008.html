Principais formatações

Negrito/Destaque

Nesta frase temos um termo em negrito usando a tag B (não semântica), que, apesar de ser utilizada no HTML4, ainda não caiu em desuso.

Nesta frase, temos um termo em destaque usando a tag STRONG (semântica).

Itálico/Enfâse

Nesta frase, temos um termo em itálico usando a TAG I (não semântica).

Nesta frase, temos um termo em enfâse, em inglês emphasis, usando a tag EM (semântica).

Podemos criar também um texto marcado usando a tag MARK (marca em inglês).

E, no outro parágrafo, temos texto marcado. Utilizando o CSS STYLE na parte HEAD, foi possível manter o destaque MARK em uma cor padrão, em todas as partes que onde foi utilizado.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo. Vale lembrar que a tag BIG, apesar de funcionar, não é mais utilizada.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Para isso é utilizada a tag DEL, de deletar.

Testo inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Para isso é utilizada a tag INS (semântica).

Existe também o sublinhado com a tag U (não semântica).

Texto sobrescrito

Para inserir coisas do tipo x20+3, será utilizada a tag SUP.

Texto subscrito

Para inserir coisas do tipo H2O, será utilizada a tag SUB.
